# Personal Assistant CLI Application

## Description

This project is a command-line Personal Assistant application written in Python.

The assistant can store and manage contacts, phone numbers, and birthdays.  
The application supports adding contacts, changing phone numbers, showing phone numbers, displaying all contacts, adding birthdays, showing birthdays, and finding upcoming birthdays.

This homework demonstrates:

- creating a Python virtual environment with Poetry;
- fixing the Python version for the project;
- managing project dependencies;
- creating a Docker image for the application;
- running the application inside a Docker container in interactive mode.

## Technologies Used

- Python 3.13
- Poetry
- Docker
- Colorama

Available Commands
add
change
phone
all
delete
add-birthday
show-birthday
birthdays
exit
